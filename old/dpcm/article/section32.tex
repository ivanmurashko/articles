\subsection{Поля падающие на кристалл  справа}

Рассмотрим теперь поле $E_2$. Опять же применим лемму Лоренца (\ref{lemmaLor}) 
для этого случая.
Область интегрирования изображена на рис. \ref{figLor2}.
 
В качестве эквивалентного тока выберем:
\begin{equation} 
\vec{j}_{eq}=i\vec{E}_2\mbox{\tiny$\Delta$\normalsize}\varepsilon
\frac{\omega}{4\pi}, 
\label{fj_r} 
\end{equation} 
где возмущение диэлектрической проницаемости
\tiny{$\Delta$}\normalsize$\varepsilon$
определяется соотношением (\ref{epsilon})

В качестве поля ($a$) в (\ref{lemmaLor}) примем поле $\vec{E}_2$ в виде
разложения (\ref{field_r}) с источником $\vec{j}_{eq}$ в виде
(\ref{fj_r}), а в качестве произвольного вспомогательного поля ($b$),
не имеющего источников, примем поочередно 
$\vec{e}_2^\ast e^{i(\vec{k}_2\vec{r})}$,
$\vec{e}_1 e^{-i(\vec{k}_1\vec{r})}$ и 
$\vec{f}_f^{(2)\ast}e^{-i(\vec{k}_1\vec{r})}$ - обращенные по отношению к
искомым полям.  
Действуя по аналогии с изложенным выше и вводя следующее обозначение 
\[
M_f^{(2)}=\left\langle
\int  \limits_S{\varepsilon_m(\vec{f}_f^{(2)}\vec{e}_2^\ast)ds} \right\rangle,
\]
 для полей справа можно записать
\begin{eqnarray}
\frac{\partial{A_s^{(2)}}}{\partial{z}} = 
 \frac{i\omega}{2 c \cos{\theta_2}}
\sqrt{\frac{\mu}{\varepsilon}} \left\{ 
A_c^{(2)}M^{\ast} + A_f^{(1)}M_f^{(2)} \right\},
\nonumber \\
\frac{\partial{A_c^{(2)}}}{\partial{z}} =  
\frac{i\omega}{2 c \cos{\theta_1}}
\sqrt{\frac{\mu}{\varepsilon}}A_s^{(2)}M,
\nonumber \\
\frac{\partial{A_f^{(2)}}}{\partial{z}} = 
\frac{i\omega}{2 c
\cos{\theta_1}}
\sqrt{\frac{\mu}{\epsilon}}A_s^{(2)}M_f^{(2)\ast}.
\label{E2}
\end{eqnarray}
