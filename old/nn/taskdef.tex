\section{Постановка задачи}

Предполагается создать математический модель нейронной сети на базе фоторефрактивных кристаллов. Нейронная сеть предстьавляет собой совокупность кристаллов, соединенных связями. Для каждого кристалла создается простейшая математическая модель которая моделирует основные свойства данных кристаллов. 

\section{Основные свойства ФРК}
Для математической модели ФРК, необходимо выделить основные свойства данных кристаллов.
\begin{itemize}
\item Записью является корреляция между двумя образами.
\item Запись производится на решетки
\item Процесс обучения и распознавания проходит одинаковым образом. Отличие заключается лишь во временных параметрах этого процесса
\end{itemize}
